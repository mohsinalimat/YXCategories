
### 一些实用的分类

Just For Myself

1. import `pod 'YXCategories'` in project's **Podfile** 
2. enter `YXCategories` directory with `terminal`
3. `pod update`
4. waiting....
5. import `YXCategories.h` or category prefix in your file

**upload cocoapods**

 1. ` pod spec lint`
 2. `pod trunk push YXCategories.podspec --allow-warnings`
